# Search for mating types 

Create a database for your mating types 1 and 2, then search your genome list with BLASTP.
> Note: if you don't have fasta protein sequences, use blastx to search nucleotide sequences against the protein database.


## Make database for MAT1-1 and MAT1-2
```bash
makeblastdb \
  -in MAT1-1_filtered_nr.fasta \
  -dbtype prot \
  -out MAT1-1_db
  ```
## Search your queries in the database
```bash
  blastp \
  -query genome_assemblies.faa \
  -db MAT1-1_db \
  -out MAT1-1_vs_Ccorticale.tsv \
  -evalue 1e-5 \
  -max_target_seqs 10 \
  -num_threads 8 \
  -outfmt "6 qseqid sseqid pident length qlen slen evalue bitscore stitle"
  ```

  Then, explore your results in .tsv (tabular separated values) format/
